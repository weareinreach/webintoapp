iOS Universal Links:
-----------------------------------------------------

- 	The generated the '.well-known/apple-app-site-association'.

-	Change the ABCDE12345.com.example to your own values, this should be:
	<Application Identifier Prefix>.<Bundle Identifier>

-	Upload the '.well-known' folder with all its content 
	(apple-app-site-association file) to the root of the hostname.

-	More information can be found at:
	https://developer.apple.com/ios/universal-links/
	https://developer.apple.com/documentation/xcode/supporting-associated-domains

Please note, in order to display the '.well-known' directory under 
the finder of the MacOS press the following keys together: 
cmd + shift + full-stop (period). That is: ⌘ + ⇧ + .
